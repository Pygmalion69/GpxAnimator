# GpxAnimator

A desktop application to animate GPX tracks on an interactive map and export the animation as an MP4 video.

### Features
- **Open GPX Files**: Load and display multiple tracks from GPX files.
- **Interactive Animation**: Play and stop the track animation directly within the application.
- **Video Export**: Capture the map animation and encode it into a high-quality MP4 video using FFmpeg.

### Requirements
- Python 3
- PyQt6
- Folium
- gpxpy
- FFmpeg (for video export)

### Usage
Run the application using:
```bash
python main.py
```
